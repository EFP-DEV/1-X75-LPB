# Exercices : Les entrées / sorties

## Exercice 2.1
Quel résultat produit ce programme ?
```
Variables val, double en Numérique

Début
    val ← 231
    double ← val * 2
    Ecrire val
    Ecrire double
Fin
```

## Exercice 2.2
Écrire un algorithme qui demande un nombre à l'utilisateur, puis calcule et affiche le carré de ce nombre.

Exemple d'exécution :
```
Entrez un nombre : 2
Le carré de 2 est 4
```

## Exercice 2.3
Un food truck vend des tacos à 8€ pièce. Écrire un algorithme qui demande le nombre de tacos commandés et affiche le total à payer.

Exemple d'exécution :
```
Combien de tacos ? 3
Total : 24€
```

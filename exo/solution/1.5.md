# Exercice 1.5

## Introduction

Dans cet exercice, tu dois **suivre l’algorithme ligne par ligne** et observer comment les variables changent.

Une variable est comme une **boîte** :

* quand on fait une **affectation** (`<-`), on met une valeur dans la boîte ;
* si on affecte une nouvelle valeur plus tard, **on remplace** l’ancienne.

---

## Objectifs

Comprendre que :

* une variable contient une valeur ;
* une affectation remplace la valeur précédente ;
* une variable n’est pas reliée automatiquement à une autre.

---

## Pseudo-code

```
1   Variables mon_age, ton_age en Entier
2
3   Début
4       mon_age <- 5
5       ton_age <- 219
6       mon_age <- ton_age
7       ton_age <- mon_age
8   Fin
```

---

## Syntaxe

* **Déclaration** : `Variables x, y en Entier`
  → On annonce les variables qu’on va utiliser, et leur type (ici : nombres entiers).

* **Affectation** : `x <- 5`
  → La flèche **<-** se lit **« prend la valeur »** / **« on met dans »**.
  Ce n’est **pas** une égalité mathématique : c’est une **commande** qui remplace la valeur précédente.

* **Utiliser une variable dans un calcul** : `y <- x + 3`
  → On prend la **valeur actuelle** de `x`, on calcule, puis on stocke le résultat dans `y`.

* **Ordre des lignes** : on exécute **de haut en bas**, une ligne après l’autre.

* **Fin de ligne** : dans ce pseudo-code, pas besoin de `;` (contrairement au C).
  Les blocs sont indiqués par `Début` / `Fin` et par l’indentation.

* **Nom des variables** : un nom est une “étiquette” (ex. `mon_age`, `ton_age`) ; il sert à retrouver la valeur stockée.

## Idée essentielle

Quand on écrit :

```
mon_age <- ton_age
```

L’ordinateur :

1. regarde la valeur actuelle de `ton_age` ;
2. copie cette valeur ;
3. la met dans `mon_age` (en remplaçant l’ancienne).

Ensuite, `mon_age` et `ton_age` ne sont pas “liés” :
ce sont juste deux boîtes qui contiennent, à ce moment-là, la même valeur.

---

## Image mentale

Une variable n’est pas un tuyau relié à une autre variable.
C’est une boîte indépendante.

Quand on copie une valeur d’une boîte vers une autre, on ne crée pas de lien : on recopie juste le contenu.

---

## Conclusion

Après exécution complète :

```
mon_age = 219
ton_age = 219
```

---

## Que s'est-il passé ?

On a voulu faire comme un “échange”, mais ça n’a **pas** échangé.

* À la ligne 6, `mon_age` prend 219 → la valeur 5 est perdue.
* À la ligne 7, `ton_age` prend la valeur de `mon_age`… qui vaut déjà 219.

Donc on se retrouve avec **deux fois 219**.

Pour échanger deux valeurs, il faut une **troisième variable** (une boîte temporaire) pour garder la première valeur avant qu’elle soit remplacée.

# Exercice 1.2

## Introduction

Dans cet exercice, tu dois **suivre l’algorithme ligne par ligne** et observer comment les variables changent. 

Une variable est comme une **boîte** :

* quand on fait une **affectation** (`<-`), on met une valeur dans la boîte ;
* si on affecte une nouvelle valeur plus tard, **on remplace** l’ancienne.

---

## Objectifs

Comprendre que : 

* une variable contient une valeur ;
* une affectation remplace la valeur précédente ;
* une variable n’est pas reliée automatiquement à une autre.

---

## Pseudo-code

```
1   Variables pommes, oranges, total en Entier
2
3   Début
4       pommes <- 5
5       oranges <- 3
6       total <- pommes + oranges
7       pommes <- 2
8       total <- oranges - pommes
9   Fin
```

---

## Syntaxe

* **Déclaration** : `Variables x, y en Entier` 
  → On annonce les variables qu’on va utiliser, et leur type (ici : nombres entiers).

* **Affectation** : `x <- 5` 
  → La flèche **<-** se lit **« prend la valeur »** / **« on met dans »**.
  Ce n’est **pas** une égalité mathématique : c’est une **commande** qui remplace la valeur précédente.

* **Utiliser une variable dans un calcul** : `y <- x + 3` 
  → On prend la **valeur actuelle** de `x`, on calcule, puis on stocke le résultat dans `y`.

* **Ordre des lignes** : on exécute **de haut en bas**, une ligne après l’autre. 

* **Fin de ligne** : dans ce pseudo-code, pas besoin de `;` (contrairement au C). 
  Les blocs sont indiqués par `Début` / `Fin` et par l’indentation.

* **Nom des variables** : un nom est une “étiquette” (ex. `pommes`, `oranges`, `total`) ; il sert à retrouver la valeur stockée.

## Idée essentielle

Quand on écrit :

```
total <- pommes + oranges
```

L’ordinateur :

1. regarde la valeur actuelle de `pommes` et de `oranges` ;
2. fait le calcul ;
3. range le résultat dans `total`.

Puis c’est terminé :
il ne garde pas de “connexion” avec `pommes` ou `oranges`.

---

## Image mentale

Une variable n’est pas un tuyau relié à une autre variable.
C’est une boîte indépendante. 

Si on change le contenu d’une boîte, cela ne modifie pas les autres boîtes (sauf si on recalcule et qu’on réécrit une valeur).

---

## Conclusion

Après exécution complète :

```
pommes = 2
oranges = 3
total = 1
```

**Parce que :**

* `total` vaut d’abord 8 (ligne 6), puis il est **réécrit** à la ligne 8 ;
* à la ligne 8, on utilise les valeurs actuelles : `oranges = 3` et `pommes = 2`, donc `3 - 2 = 1`.

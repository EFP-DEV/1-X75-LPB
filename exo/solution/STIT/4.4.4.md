# Exercice 4.4 — Jeu de tests 

## Objectif des tests

Vérifier que l’algorithme respecte exactement les règles :

* Draxx : exclusion si `age >= 1835`
* Sklounst : exclusion si `age >= 18` et `age < 35`
* Tous les autres : autorisé

On teste donc :

* les **3 issues** possibles (exclu Draxx / exclu Sklounst / autorisé),
* toutes les **bornes** : `1835`, `18`, `35`,
* les valeurs juste avant / juste après,
* et des genres “autres”.

---

## Format attendu

Pour chaque test :

* Entrée : `genre`, `age`
* Sortie attendue : `"Peut voter."` ou `"Ne peut pas voter."`

---

# Jeu de tests

## A — Draxx (borne 1835)

**T01 — Draxx juste en-dessous**

* Entrée : `genre="Draxx"`, `age=1834`
* Attendu : **Peut voter.**

**T02 — Draxx à la borne**

* Entrée : `genre="Draxx"`, `age=1835`
* Attendu : **Ne peut pas voter.**

**T03 — Draxx au-dessus**

* Entrée : `genre="Draxx"`, `age=2000`
* Attendu : **Ne peut pas voter.**

---

## B — Sklounst (bornes 18 et 35)

**T04 — Sklounst juste en-dessous de 18**

* Entrée : `genre="Sklounst"`, `age=17`
* Attendu : **Peut voter.**

**T05 — Sklounst à 18 (18 compris)**

* Entrée : `genre="Sklounst"`, `age=18`
* Attendu : **Ne peut pas voter.**

**T06 — Sklounst dans l’intervalle**

* Entrée : `genre="Sklounst"`, `age=34`
* Attendu : **Ne peut pas voter.**

**T07 — Sklounst à 35 (35 non compris)**

* Entrée : `genre="Sklounst"`, `age=35`
* Attendu : **Peut voter.**

**T08 — Sklounst au-dessus de 35**

* Entrée : `genre="Sklounst"`, `age=36`
* Attendu : **Peut voter.**

---

## C — Autres genres (contrôle)

**T09 — Autre genre, petit âge**

* Entrée : `genre="Glorbz"`, `age=5`
* Attendu : **Peut voter.**

**T10 — Autre genre, grand âge**

* Entrée : `genre="Glorbz"`, `age=5000`
* Attendu : **Peut voter.**

---

## D — Pièges (priorité des règles)

Ces tests vérifient qu’on n’exclut pas “par accident” un genre qui n’est pas concerné.

**T11 — âge 1835 mais pas Draxx**

* Entrée : `genre="Sklounst"`, `age=1835`
* Attendu : **Peut voter.**
  (car la règle Sklounst s’arrête avant 35)

**T12 — âge 18 mais pas Sklounst**

* Entrée : `genre="Draxx"`, `age=18`
* Attendu : **Peut voter.**
  (car Draxx n’est exclu qu’à partir de 1835)

---

## Résumé des couvertures

* **Draxx** : 1834 / 1835 / au-dessus 
* **Sklounst** : 17 / 18 / 34 / 35 / 36 
* **Autres genres** : contrôle 
* **Pièges** : âges “sensibles” avec genre non concerné

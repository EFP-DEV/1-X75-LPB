# Exercice 3.3 — Version 2 (condition complète)

## Introduction

Dans cette version, on corrige le défaut de la version 1 : pour que trois noms soient triés, il ne suffit pas de vérifier les deux premiers.

Il faut que l’ordre soit respecté entre :

* le 1er et le 2e
* le 2e et le 3e

Les noms identiques sont considérés comme triés, donc on utilise `<=`.

---

## Objectifs

Comprendre que :

* “trié” signifie : `n1 <= n2` et `n2 <= n3`
* on doit combiner deux conditions avec `ET`
* `<=` permet d’accepter les égalités

---

## Pseudo-code

```
1   Variables n1, n2, n3 en Chaîne
2
3   Début
4       Ecrire "Entrez le premier nom : "
5       Lire n1
6       Ecrire "Entrez le deuxième nom : "
7       Lire n2
8       Ecrire "Entrez le troisième nom : "
9       Lire n3
10
11      Si (n1 <= n2) ET (n2 <= n3) Alors
12          Ecrire "Les noms sont triés."
13      Sinon
14          Ecrire "Les noms ne sont pas triés."
15      FinSi
16  Fin
```

---

## Syntaxe

* `<=` : “avant ou égal” (dans l’ordre alphabétique).
* `ET` : les deux conditions doivent être vraies.
* `( ... )` : clarifie le regroupement des comparaisons.

---

---

## Inspection des valeurs

### Exemple 1 — n1 = "Alice", n2 = "Bernard", n3 = "Charles"

* `n1 <= n2` : vrai
* `n2 <= n3` : vrai
* condition complète : vrai
  Affichage : Les noms sont triés.

### Exemple 2 — n1 = "Alice", n2 = "Charles", n3 = "Bernard"

* `n1 <= n2` : vrai
* `n2 <= n3` : faux
* condition complète : faux
  Affichage : Les noms ne sont pas triés.

### Exemple 3 — n1 = "Alice", n2 = "Alice", n3 = "Bernard"

* `n1 <= n2` : vrai (égalité acceptée)
* `n2 <= n3` : vrai
* condition complète : vrai
  Affichage : Les noms sont triés.

---

## Amélioration par rapport à la version 1

Version 1 : ne testait que `n1 <= n2`, donc pouvait se tromper.
Version 2 : teste `n1 <= n2` et `n2 <= n3`, ce qui est la condition correcte pour trois noms.

---

## Conclusion

Avec la condition :

`(n1 <= n2) ET (n2 <= n3)`

l’algorithme annonce “triés” uniquement quand les trois noms respectent l’ordre alphabétique, en acceptant les noms identiques.

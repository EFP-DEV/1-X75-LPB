# Exercice 2.4

## Introduction

Dans cet exercice, tu dois écrire un algorithme qui :

1. **demande** le prix HT d’un article ;
2. **demande** la quantité ;
3. **demande** le taux de TVA ;
4. **calcule** le total TTC ;
5. **affiche** le résultat.

Une variable est comme une **boîte** :

* quand on fait une **affectation** (`<-`), on met une valeur dans la boîte ;
* `Lire` met une valeur dans une variable ;
* `Ecrire` affiche, **sans modifier** les variables.

---

## Objectifs

Comprendre que :

* le total HT se calcule avec une multiplication ;
* la TVA s’applique comme un coefficient ;
* on peut afficher un total final TTC.

---

## Pseudo-code

```
1   Variables prix_ht, quantite, taux_tva, total_ht, total_ttc en Numérique
2
3   Début
4       Ecrire "Prix HT ? "
5       Lire prix_ht
6
7       Ecrire "Quantité ? "
8       Lire quantite
9
10      Ecrire "Taux de TVA (%) ? "
11      Lire taux_tva
12
13      total_ht <- prix_ht * quantite
14      total_ttc <- total_ht * (1 + taux_tva / 100)
15
16      Ecrire "Total TTC : ", total_ttc
17  Fin
```

---

## Syntaxe

* **Lire** : `Lire prix_ht`
  → Met la valeur saisie dans la variable.

* **Total HT** : `total_ht <- prix_ht * quantite`
  → Prix unitaire HT × quantité.

* **Passer de HT à TTC** : `total_ttc <- total_ht * (1 + taux_tva / 100)`
  → Si la TVA est 21%, on multiplie par `1 + 21/100 = 1,21`.

* **Afficher** : `Ecrire "Total TTC : ", total_ttc`
  → Affiche le total.

## Idée essentielle

On applique la TVA en multipliant par :

```
1 + (taux_tva / 100)
```

Exemples :

* 6%  → coefficient 1,06
* 21% → coefficient 1,21

---

## Image mentale

Tu calcules d’abord le prix **sans TVA** (HT), comme au dépôt.
Puis tu ajoutes la TVA en multipliant, comme au passage en caisse.

---

## Conclusion

L’algorithme :

1. lit `prix_ht`, `quantite`, `taux_tva` ;
2. calcule `total_ht = prix_ht * quantite` ;
3. calcule `total_ttc = total_ht * (1 + taux_tva/100)` ;
4. affiche le total TTC.

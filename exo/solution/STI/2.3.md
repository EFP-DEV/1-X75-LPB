# Exercice 2.3

## Introduction

Dans cet exercice, tu dois écrire un algorithme qui :

1. **demande** combien de tacos sont commandés ;
2. **calcule** le total à payer ;
3. **affiche** ce total.

Une variable est comme une **boîte** :

* quand on fait une **affectation** (`<-`), on met une valeur dans la boîte ;
* `Lire` met une valeur dans une variable ;
* `Ecrire` affiche, **sans modifier** les variables.

---

## Objectifs

Comprendre que :

* on peut lire une quantité saisie par l’utilisateur ;
* on peut calculer un prix total avec une multiplication ;
* on peut afficher un résultat clair.

---

## Pseudo-code

```
1   Variables nb_tacos, total en Numérique
2
3   Début
4       Ecrire "Combien de tacos ? "
5       Lire nb_tacos
6       total <- nb_tacos * 8
7       Ecrire "Total : ", total, "€"
8   Fin
```

---

## Syntaxe

* **Lire une valeur** : `Lire nb_tacos`
  → Récupère le nombre saisi par l’utilisateur et le met dans `nb_tacos`.

* **Calculer** : `total <- nb_tacos * 8`
  → Multiplie le nombre de tacos par 8 (prix d’un taco) et met le résultat dans `total`.

* **Afficher** : `Ecrire "Total : ", total, "€"`
  → Affiche un texte, puis la valeur calculée.

## Idée essentielle

Le total à payer se calcule avec :

```
total = quantité * prix_unitaire
```

Ici :

* quantité = `nb_tacos`
* prix_unitaire = 8

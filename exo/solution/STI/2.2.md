# Exercice 2.2

## Introduction

Dans cet exercice, tu dois écrire un algorithme qui :

1. **demande** un nombre à l’utilisateur ;
2. **lit** ce nombre ;
3. **calcule** son carré ;
4. **affiche** le résultat.

Une variable est comme une **boîte** :

* quand on fait une **affectation** (`<-`), on met une valeur dans la boîte ;
* `Lire` met une valeur dans une variable ;
* `Ecrire` affiche, **sans modifier** les variables.

---

## Objectifs

Comprendre que :

* `Lire` permet de récupérer une valeur saisie par l’utilisateur ;
* on peut calculer une nouvelle valeur à partir d’une variable ;
* on peut afficher un résultat avec `Ecrire`.

---

## Pseudo-code

```
1   Variables n, carre en Numérique
2
3   Début
4       Ecrire "Entrez un nombre : "
5       Lire n
6       carre <- n * n
7       Ecrire "Le carré de ", n, " est ", carre
8   Fin
```

---

## Syntaxe

* **Afficher un message** : `Ecrire "Entrez un nombre : "`
  → Affiche une phrase à l’écran.

* **Lire une valeur** : `Lire n`
  → Récupère ce que l’utilisateur tape et le met dans `n`.

* **Calculer** : `carre <- n * n`
  → Calcule `n` fois `n`, puis range le résultat dans `carre`.

* **Afficher plusieurs éléments** : `Ecrire "Le carré de ", n, " est ", carre`
  → Affiche un texte + des valeurs dans la même ligne.

## Idée essentielle

Le carré d’un nombre `n` se calcule en faisant :

```
n * n
```

Ce n’est pas un “nouveau nombre magique” :
c’est juste une multiplication.

---

## Image mentale

Tu mets un nombre dans la boîte `n`.
Tu multiplies ce nombre par lui-même.
Tu ranges le résultat dans la boîte `carre`.
Puis tu l’annonces à l’écran.

---

## Conclusion

L’algorithme :

1. demande un nombre ;
2. calcule `n * n` ;
3. affiche :
   `Le carré de n est carre`.
